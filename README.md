## How to use it

> [!NOTE]  
> **1** - Clone the repository by running:  

```bash
git clone https://github.com/sdkitagawa/kiro-swe-codex.git
```

> [!TIP]  
> **2** - Choose a Roadmap according to your knowledge level.  
> **3** - Start learning.  

## How to Contribute

> [!IMPORTANT]  
> You can contribute to the project in several different ways:  
> 
> - [**Translate Existing Content**](./CONTRIBUTING.md#translate-existing-content): You can fork this repository, navigate to the `roadmap` folder and then go to the folder containing the content you want to translate and add your translation.  
> - [**Suggest New Content**](./CONTRIBUTING.md##suggest-new-content): If you want, you can suggest that we add new content to the Roadmap by opening issues.  
>
